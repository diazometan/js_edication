## Учебный проект в ЯндексПрактикуме

### Описание

Сервис обмена карточками интересных мест.

### Технологии

Стек:
* HTML
* CSS
* Нативный JavaScript

Для сборки проекта использован Webpack:
* оптимизированы картинки и шрифты
* минимизация CSS кода 
* добавлены вендорные префиксы
* транспиляция JS-кода в Babel

### Использование

Демо проекта: https://diazometan.github.io/js_edication/

Для запуска проекта локально:

- `git clone https://github.com/diazometan/js_edication.git` - клонировать репозиторий
- `npm install` - установить все необходимые пакеты
- `npm run dev` - проект будет откроется в браузере на порте 8080
